#  Mood Music App

A Flutter-based mobile application that recommends songs based on the user's current mood. Users can select moods such as Happy, Sad, Angry, and Relax, and instantly receive relevant song suggestions. Songs can be played through YouTube using external links.

## Features
* Mood-based song recommendations
* Simple and attractive user interface
* Happy, Sad, Angry, and Relax mood categories
* Direct song playback through YouTube
* Fast and smooth navigation
* Built using Flutter and Dart

## Technologies Used
* Flutter
* Dart
* Android Studio
* url_launcher Package
* Git & GitHub

## 🚀 Installation

1. Clone the repository
```bash
git clone https://github.com/Ezilarasi2006/mood_music_app.git
```
2. Navigate to project folder
```bash
cd mood_music_app
```
3. Install dependencies
```bash
flutter pub get
```
4. Run the application
```bash
flutter run
```
##  How It Works
1. User opens the application.
2. Selects a mood.
3. App displays songs related to that mood.
4. Clicking a song opens YouTube and plays the selected song.
